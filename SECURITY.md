# Security Policy

## Supported Versions

AIHAT is currently in active pre-release development.

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Older pre-release versions | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Use GitHub private vulnerability reporting when available for this repository.

Include:

- A clear description of the vulnerability
- The affected AIHAT version or commit
- Steps to reproduce the issue
- Potential impact
- Any suggested mitigation

Reports will be reviewed as soon as practical. Please allow reasonable time for investigation and remediation before public disclosure.

## Scope

Security reports may include:

- Credential or secret exposure
- Unsafe command execution
- Privilege escalation risks
- Insecure file or registry handling
- Improper permissions
- Data leakage
- Vulnerable third-party dependencies
